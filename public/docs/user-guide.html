1. 앱 개요 및 시작하기

앱이 제공하는 기능

  • 자산/연금/부채 등록 및 현황 집계
  • 수입/지출 기록 및 월별 정산(마감)
  • 목적자금 저축 목표 추적 (진행률 시각화)
  • 통합 대시보드 — 순자산 추이, 카테고리 비중 차트
  • 자산 시뮬레이션 — 교육비·포트폴리오 기반 미래 자산 예측
  • AI 재무 상담 — 내 재무 데이터를 기반으로 맞춤 조언

로그인

Microsoft 또는 GitHub 계정으로 로그인합니다. 설정 페이지 상단의 로그인 버튼을 사용하세요.

13. 정산 시스템 안내

수입과 지출 페이지에는 월별 정산(마감) 기능이 있습니다.

정산이란?

매월 마감 시 고정 템플릿으로 등록한 항목(월급, 구독료, 투자이체 등)이 해당 월의 실적 데이터로 자동 생성됩니다.

정산 흐름

1. 정산 월 선택
예: 2025-02
→
2. "마감" 버튼 클릭
고정 항목 자동 생성
→
3. ✅ 정산 완료
유동자산 자동 반영

수입 정산

  • "고정수입 템플릿" 체크된 항목이 해당 월의 수입으로 자동 생성
  • "유동자산 반영" 체크된 항목은 현금 자산에 자동 가산

지출 정산

  • 카드 미포함 고정/구독 지출이 해당 월의 실적으로 자동 생성
  • 투자이체 항목도 자동 생성
  • "유동자산 반영" 체크된 항목은 현금 자산에서 자동 차감

정산 취소 (롤백)

정산된 월에는 "롤백" 버튼이 표시됩니다. 롤백 시 해당 월의 자동 생성된 항목이 삭제되고, 유동자산 반영분도 원복됩니다.

⚠️ 주의 정산 후 수동으로 추가한 항목은 롤백 시 함께 삭제되지 않습니다. 정산 전에 일회성/변동 항목을 먼저 수동 등록하는 것을 권장합니다.

FAQ

Q: 연금은 왜 자산 페이지에 안 보이나요?

A: 연금은 연금관리 메뉴에서 별도 관리됩니다. 유동성이 낮아 일반 자산과 분리했습니다. 홈/대시보드에서는 "연금 포함" 합계로 함께 볼 수 있습니다.

Q: 카드값 간편입력과 일반 지출의 차이는?

A: 카드값 간편입력은 카드사와 결제액만 입력하면 모든 설정이 자동 완료됩니다. 일반 지출은 카테고리, 주기, 고정/구독 여부 등 세밀한 설정이 가능합니다.

Q: AI 상담에서 내 데이터를 어떻게 활용하나요?

A: 상담 시작 시 프로파일, 자산, 지출 요약 등을 자동으로 AI에게 전달합니다. 개인정보는 OpenAI API를 통해 처리되며 별도 저장되지 않습니다.

Q: 자산 스냅샷은 어떻게 기록되나요?

A: 매월 말일 정오에 Azure Functions의 타이머 트리거가 실행되어 전체 자산 평가액을 자동으로 기록합니다. 사용자가 별도로 실행할 필요 없습니다.

Q: 목적자금과 지출은 어떻게 연동되나요?

A: 지출 페이지의 "투자·저축 이체" 탭에서 목적자금을 연결하면, 정산 시 해당 금액이 목적자금의 월별 적립 기록에 자동 반영됩니다.
